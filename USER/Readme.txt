
桌面机器人项目计划

一、硬件资源：
	1、激光测距。
	2、I2C控制3731-双眼灯板。
	3、蓝牙透传模块、UART平板通信、UART调试。
	4、电机控制。
	5、按键、LED状态显示、三色LED。
	6、自动关机
	
二、调试中遇到的问题：
	1、操作系统uCOS需要每个中断都要添加  OSIntEnter();    OSIntExit();CAN中断和
串口UART中断都需要添加。
	2、串口UART具有FFIO缓冲区，使用的时候要关闭FIFO，才会收到一个数据就中断。
	3、申请内存块，用完后一定要释放。
	4、移植操作系统，最关键的是时钟嘀嗒和堆栈增长方向。
	5、蓝牙波特率：  115200
	6、内存方案是：谁用谁释放，切记不要内存泄漏。
